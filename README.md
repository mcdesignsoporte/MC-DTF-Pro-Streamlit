# MC DTF Pro Streamlit

Versión web simple y gratuita para Streamlit Community Cloud.

## Funciones

- Eliminación de fondo con IA.
- Limpieza de semitransparencias.
- Limpieza de píxeles basura.
- Presets: Rápido, Calidad, Logo fuerte, PNG transparente.
- Exportación PNG.
- Exportación PDF opcional.
- Semitonos opcionales.
- Vista previa sobre fondo negro, blanco y rojo.

## Deploy gratis

1. Sube estos archivos a GitHub.
2. Entra a https://share.streamlit.io/
3. Conecta el repositorio.
4. Main file path: `streamlit_app.py`
5. Deploy.

## Uso local opcional

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
